# Todo App (React)

Простое и удобное приложение для управления списком задач, созданное с использованием **React**.  
Позволяет добавлять, отмечать, редактировать и удалять задачи, помогая организовать повседневные дела.

## Возможности

- Добавление новых задач  
- Отметка задач как выполненных  
- Удаление задач
- Редактирование задач

## Технологии

- **React**
- **JavaScript**
- **HTML5**
- **CSS3**

## Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/annaivastudent/react-todo-app.git

2. Перейди в папку проекта:
```bash
    cd todo-app
```

3. Установи зависимости:
```bash
npm install
```

4. Запусти приложение:
```bash
npm start
```
